# Tiers & Proposal Structure

## Tier definitions

| | **Starter — $2,500/mo** | **Core — $5,000/mo** | **Embedded — $8,000+/mo** |
|---|---|---|---|
| Workflows automated | 1–2 | Up to 4 ("unlimited agents" within them) | Ongoing portfolio across the business |
| Improvement cycle | Monthly (1 improvement) | Monthly + 1 new workflow/quarter | Monthly + quarterly roadmap session |
| Monitoring | Business-hours alerting | 24/7 watchdogs, priority fixes | 24/7, same-day fix SLA |
| Reporting | Weekly report | Weekly + monthly value summary | Weekly + monthly + quarterly business review |
| Access to operator | Async only | Async + one 30-min call/month | Fractional AI-officer; up to 2 calls/month |
| **Excludes** | New workflows mid-cycle; calls; custom integrations | Custom software dev; >1 new workflow/quarter; daily calls | Full-time availability; work outside roadmap; regulated-data workflows |

## Tier-fit rules

1. Count of scoped workflows → base tier.
2. +1 tier if ANY of: client requires meeting cadence · custom/unusual
   integration in the stack · micromanager red flag.
3. Estimated monthly value must be ≥3× retainer. Below that: narrow scope or
   decline. **Never discount below $2,500.**

## Proposal structure

1. Their problem, in their own intake words (3 sentences max)
2. The AI Employee Job Description (from agency-onboarding intake-system)
3. How it works: install timeline including shadow week ("training")
4. Tier + price + what's explicitly excluded
5. What happens month 2+: the improvement cycle (why it's a retainer, not a project)
6. Proof: one anonymized result or relevant workflow example
7. Next step + 14-day validity + slot scarcity (true: max 1 install/month)
